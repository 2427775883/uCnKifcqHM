# 项目简介
## 核心功能
- 用户系统
- 教学资源管理
- 课程信息系统
- 设备管理系统
- 交流论坛
- 实验数据管理
- 教育资讯
- 数据可视化大屏
- 管理后台
- 网络爬虫
## 适用场景
- 教育机构内部数据共享与交流
- 教学资源统一管理
- 课程信息查询与选课
- 设备状态监控与统计
- 学生互动交流平台
- 实验数据记录与分析
- 教育资讯获取与分类
- 数据可视化展示
## 技术特点
- 前端：Vue 3 + Element Plus + ECharts + Vite
- 后端：Python Flask + JWT + SQLAlchemy
- 数据库：MySQL 8.0
- 架构：前后端分离 + RESTful API
